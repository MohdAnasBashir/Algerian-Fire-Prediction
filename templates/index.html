Algerian Forest Fire Prediction System

This project is a Machine Learning-based web application that predicts the Fire Weather Index (FWI), an important indicator of forest fire intensity and risk. The model is trained on real-world datasets from two regions of Algeria: Bejaia (Northeast) and Sidi Bel Abbes (Northwest). It analyzes environmental and meteorological parameters such as temperature, humidity, wind speed, rainfall, and fire indices to accurately estimate FWI values.

Factors Considered

  • Temperature (°C)
  • Relative Humidity (RH %)
  • Wind Speed (km/h)
  • Rain (mm)
  • FFMC (Fine Fuel Moisture Code)
  • DMC (Duff Moisture Code)
  • ISI (Initial Spread Index)
  • Classes (Fire/No Fire)
  • Region (Bejaia / Sidi Bel Abbes)

Model Used

The prediction model is built using Linear Regression, trained on historical forest fire data to estimate Fire Weather Index (FWI) values based on environmental inputs.

Tech Stack

  • Python
  • Flask (Backend)
  • Scikit-learn (Machine Learning)
  • Pandas & NumPy (Data Processing)
  • HTML & CSS (Frontend)
